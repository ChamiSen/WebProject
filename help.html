    Usage

    On the main page, you can add new goodies and change baskets. You add a goody and then you confirm to really add it. You can also choose from one of the preconfigured baskets and goodies. Once you have your prefered goody configuration, you can save the state with a prefered name. After being saved, they can be viewed on the file page.

    The file page consists of uploading files, then loading and downloading. It also has a special feature of drag and drop section. The extension for files to be uploaded has to be .txt and it is being checked.
